Project Overview

This project focuses on improving a machine learning model that predicts T-cell receptor efficacy using features derived from molecular dynamics simulations, including contacts, correlations, hydrogen bonds, and future salt bridge features.

Timeline

June 2026

Started understanding the project background

Read through project posters and prior materials to understand the biological motivation: predicting which TCRs may be most effective at reducing tumor size.

June 2026

Explored Stella’s GitHub repository

Reviewed the previous model pipeline, including how contact, correlation, and hydrogen bond features were organized and used for classification.

June 2026

Reproduced the baseline model

Worked with XGBoost and related machine learning models to reproduce prior predictions and better understand thresholds, labels, features, and performance metrics.

Next steps
